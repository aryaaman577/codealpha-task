## ⚠️ Before You Start

### 1. Check Node.js is installed:
```
node --version
```
Should show: v14.0.0 or higher

**Don't have Node.js?** Download: https://nodejs.org/

### 2. MongoDB Setup:

**Option A: MongoDB Atlas (Easy - Recommended)**
- No installation needed
- Free cloud database
- Follow: `WINDOWS_SETUP.md` → Step 4 → Option A

**Option B: Local MongoDB**
- Download: https://www.mongodb.com/try/download/community
- Install with default settings
- That's it!

## 🆘 Having Problems?

### Common Issues:

#### "npm command not found"
→ Install Node.js from nodejs.org

#### "Cannot connect to MongoDB"
→ Read `WINDOWS_SETUP.md` Step 4

#### "Port 3000 already in use"
→ Change PORT in `.env` to 5000

#### PowerShell script errors (Windows)
→ Use Command Prompt (CMD) instead!
